## 🗞️ AI News Workflow

안녕하세요! 아침마다 쏟아지는 방대한 IT/AI 뉴스들을 하나하나 찾아보기 힘드셨죠? ☕️
이제 커피 한 잔 마시는 짧은 시간 동안, AI가 요약해준 핵심 뉴스만 쏙쏙 골라보며 하루를 스마트하게 시작해 보세요!

이 프로젝트는 RSS 피드를 통해 최신 뉴스를 수집하고, **Google Gemini AI**가 이를 멋지게 요약해서 마크다운 문서로 만들어주는 자동화 도구입니다.

*이 프로젝트는 옵시디언 환경에 맞는 md 파일로 저장됩니다.*

---

### ✨ 기능

- **자동 소식통**: 설정해둔 RSS 피드에서 최신 기술 뉴스를 자동으로 긁어옵니다.
- **AI 요약 비서**: Gemini가 뉴스를 분석해 핵심 포인트와 인사이트를 담은 마크다운 노트를 작성합니다.
- **스마트 동기화**: (선택 사항) 로컬의 다른 저장소나 GitHub 레포지토리로 자동 동기화하여 지식 베이스를 풍성하게 채워줍니다.

---

### 🛠️ 설정법

어렵지 않아요! 딱 3가지만 준비하면 끝입니다.

#### 1. 라이브러리 설치
터미널을 열고 아래 명령어를 입력해 주세요.
```bash
pip install feedparser python-dotenv google-generativeai
```

#### 2. 환경 설정 (`.env`)
`.env.example` 파일을 복사해서 `.env` 파일을 만들고, 발급받은 **Google API Key**와 파일이 저장될 **경로**만 적어주면 됩니다.

```env
# Google Gemini API 키
GOOGLE_API_KEY=여러분의_API_키

# 파일이 저장될 로컬 경로 (예: 지식 관리 폴더)
BASIC_PATH=/Users/username/MyMemo

# 상세 저장 폴더 이름 (예: 001.Daily/News)
NEWS_SUBDIR=001.Daily/AI_News
```

### 🚀 실행

준비가 끝났다면 이제 실행만 하면 됩니다. 

**스크립트로 한 번에!**
```bash
chmod +x run.sh
./run.sh
```

**혹은 파이썬으로 직접!**
```bash
python3 main.py
```

잠시만 기다리면 지정한 폴더에 AI가 깔끔하게 정리한 뉴스 파일들이 생성됩니다. 🎁

---

### 💡 팁

- **지식 관리 도구와 연계**: Obsidian(옵시디언)이나 Notion 같은 도구의 폴더를 저장 경로로 지정하면 나만의 AI 신문사가 탄생합니다!
- **GitHub 연동**: `BASIC_PATH`를 로컬 Git 저장소로 지정하면 자동으로 커밋과 푸시까지 해줍니다. (연동 설정을 하지 않으면 로컬 폴더에만 안전하게 보관됩니다!)
- **내 맘대로 템플릿**: `makeMD.md` 파일을 수정해서 AI가 작성해주는 노트의 형식을 자유롭게 커스텀할 수 있습니다.
